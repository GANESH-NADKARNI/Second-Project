# 📘 Code Magazine

A modern blog website dedicated to teaching web development fundamentals, starting with HTML.

## Overview

**Code Magazine** is an educational blog project built with pure HTML and CSS. It serves as a learning resource for aspiring web developers who want to understand the basics of web technologies.

## Features

### Pages
- **Home (index.html)** - Main blog post about HTML fundamentals
  - Detailed explanation of what HTML is
  - Why HTML is essential for web development
  - Overview of the three fundamental web technologies (HTML, CSS, JavaScript)
  
- **Blog (blog.html)** - Blog section with navigation back to home

### Navigation
- Main header navigation with links to:
  - Blog posts
  - Challenges section
  - Flexbox learning resources
  - CSS Grid learning resources

### Design & Styling
- Modern, professional design with a teal/cyan color scheme
- Glassmorphism effects with blur filters
- Responsive grid layout (1200px container)
- Flexbox and CSS Grid implementations
- Interactive link states and hover effects
- Drop shadow effects on images for visual depth
- Clean typography and proper spacing

## Technology Stack

- **HTML** (54.9%) - Semantic markup for content structure
- **CSS** (45.1%) - Modern styling with Grid, Flexbox, and visual effects

## Key CSS Features

- **CSS Grid Layout** - Two-column layout for main content and sidebar
- **Flexbox** - Navigation and card layouts
- **Pseudo-elements** - Creative styling with `::before`, `::after`, and `::first-letter`
- **Filters & Effects** - Drop shadows and blur effects
- **Responsive Design** - Clean, centered container approach

## Getting Started

To view this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/GANESH-NADKARNI/Second-Project.git
   cd Second-Project
   ```

2. Open `index.html` in your web browser to view the main blog page

3. No build tools or dependencies required - just open and view!
